## ğŸ“¦ **Installation**

**Prerequisites:**

* PHP 7.4 or higher (recommended: 8.1)
* MySQL 5.7 or higher (recommended: 8.0)
* Web server (Apache or Nginx)
* [Detailed Installation Guide](https://docs.suitecrm.com/admin/installation/)

**Steps:**

1. **Download:** Download the latest SuiteCRM release from [GitHub](https://github.com/salesagility/SuiteCRM/releases).
2. **Extract:** Extract the downloaded archive to your web server's document root.
3. **Database Setup:** Create a new MySQL database and user.
4. **Configuration:** Configure `config.php` with your database credentials. See the [installation guide](https://docs.suitecrm.com/admin/installation/) for details.
5. **Install:** Access your SuiteCRM installation via your web browser and follow the on-screen instructions.

## ğŸ› ï¸ **Configuration**

SuiteCRM uses a `config.php` file for primary configuration.  You'll also need to set up a `.env` file for various environment-specific settings.  See the [documentation](https://docs.suitecrm.com/admin/configuration/) for detailed explanations.

**Example `.env` file (`.env.example` is provided):**

```dotenv
DATABASE_DRIVER=mysql
DATABASE_HOST=localhost
DATABASE_NAME=suitecrm
DATABASE_USER=suitecrmuser
DATABASE_PASSWORD=suitecrmpassword
```
